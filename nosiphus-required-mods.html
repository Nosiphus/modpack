The Nosiphus Required Mods are designed for Minecraft 1.5.2, and are available via our MultiMC installation method.
The Nosiphus Required Mods can run on Java 8, and you can install that from Oracle's website. However, these mods were built during the heyday of Java 7, and if you want to set up a server, you will need to install that version from Oracle's archives. You'll need an Oracle account to download the archived releases. You can watch the video below to see how to install any of our modpacks.
